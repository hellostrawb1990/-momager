# Momager — Basketball

A simple, local stat tracker for one player's basketball games. Built for
parents and coaches who want to record points, rebounds, assists and shooting
% without any of the clutter.

## Your data stays on your device

- No accounts, no sign-up
- No servers — this is a static, client-side app
- No analytics, no ads, no third-party scripts
- Everything you enter (players, games, stats) is saved only in your
  browser's local storage, on your device
- Works fully offline once loaded — nothing is sent anywhere

## How to use it

1. Add a player once (name + team)
2. Start a game → tap stat buttons as they happen
3. End game → save the box score
4. View game history and season stats any time
5. Share a box score image straight from the app

## Code

Plain HTML/CSS/JavaScript, no build step, no dependencies. Everything in
this folder is what runs in your browser — nothing hidden, nothing fetched
from elsewhere.
